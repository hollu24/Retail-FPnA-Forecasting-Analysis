# 📊 Retail Sales Forecasting & FP&A Analysis Project

This project explores retail sales data to understand business performance, forecast future sales, and analyze what drives revenue changes over time. It was built to simulate real-world financial planning work done in FP&A teams.

The focus is not just on prediction, but on understanding the *why behind the numbers*—which is what makes financial analysis useful in practice.

---

## 🧠 Project Overview

In this project, I worked with a multi-store retail dataset containing sales, promotions, holidays, and external factors.

The goal was to:
- Forecast future sales using historical trends
- Understand how promotions and seasonality affect revenue
- Compare actual performance vs forecast
- Build insights that support business decision-making

This mirrors real work in financial planning and analysis roles, where data is used to support budgeting and forecasting decisions.

---

## 📁 Dataset

Source: Kaggle – Store Sales Forecasting Dataset  
https://www.kaggle.com/datasets/tanayatipre/store-sales-forecasting-dataset

The dataset includes:
- Daily sales across multiple stores
- Promotion information
- Holiday events
- Store metadata
- External economic indicators (e.g. oil prices)

## 🔍 Project Workflow

### 1. Data Cleaning & Preparation
Before analysis, the dataset was cleaned and structured for time series modeling.

- Handled missing values
- Converted date columns to proper time formats
- Merged multiple datasets (sales, stores, holidays)
- Created time-based features (month, week, season)

---

### 2. Exploratory Data Analysis (EDA)

Key patterns were explored to understand business behavior:

- Sales trends over time
- Impact of holidays on revenue
- Store-level performance differences
- Effect of promotions on sales spikes

📊 Example Insight:
Sales consistently increased during holiday periods, showing strong seasonality in customer demand.

---

### 3. Sales Forecasting Model

A forecasting approach was built using historical trends and seasonality.

Key steps:
- Established baseline sales trend
- Incorporated seasonal patterns (holidays, weekends)
- Compared forecasted values with actual sales

---

### 4. Variance Analysis (FP&A Approach)

One of the most important parts of the project was comparing forecast vs actual performance.

This helped identify:
- Periods of overperformance and underperformance
- The impact of promotions on revenue deviations
- Stores with consistent forecast errors

This approach reflects real-world financial planning and analysis work, where variance analysis is used to explain business performance.

## 💡 Key Insights

- Sales are strongly seasonal, with clear spikes during holidays  
- Promotions significantly influence short-term revenue increases  
- Some stores consistently outperform others due to location and demand patterns  
- Forecast accuracy improves when external factors are included
